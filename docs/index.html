DB 형상 비교

SQL Server 프로시저·함수 정의를 환경(PRD, STG, DEV, QA)끼리 비교하고, 공통코드 CM_CD_D 갭도 엑셀로 뽑을 수 있는 도구입니다.

이 사이트는

GitHub Pages는 정적 페이지만 호스팅합니다. Streamlit UI는 PC에서 streamlit run app.py로 실행하세요. 자세한 절차는 저장소 README를 보시면 됩니다.

빠른 시작 (로컬)

python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env        # 접속 정보 입력
streamlit run app.py
